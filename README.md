## 🎯 The Architecture
A scalable, high-performance grading engine built to replace manual data entry. It processes raw academic data, calculates exact percentages, assigns strict grades, and generates real-time macro-analytics for top-level management.

## ⚡ Core Capabilities
- **Automated Math Engine:** Calculates exact percentages across 7 subjects and assigns strict standardized grades.
- **Macro Analytics Command Center:** Real-time dashboard computing system-wide pass rates and total processed records.
- **The Elite Leaderboard:** Dynamic ranking system highlighting top-performing assets automatically.
- **Secure Vault:** Role-based access preventing unauthorized entry into the grading and directory routes.

## 🛠️ Tech Stack
- **Backend Protocol:** Python, Flask, SQLAlchemy ORM
- **Frontend Interface:** HTML5, Custom CSS, Bootstrap 5 (Premium Dark UI)
- **Database Operations:** MySQL / SQLite Integration
- **Version Control:** Git & GitHub
